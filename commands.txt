
SNMP required api :-

$ snmpget -v 1 -c public agent_IP_ADDRESS OID_number

-v : version
-c : community string it is public
agent_IP_Address : host ip address to which we want information about various part
OID_number       : hierarchical number as 1.3.6.1. to define various part as memory , network , fan etc

$sudo tcpdump -XX -i eth0 =>to display the packet in hex format 
$sudo tcpdump -A  -i eth0 =>to display the packet in Ascii format
$sudo tcpdump -w 0001.pcap -i eth0 => to capture and save packet in the file
$sudo tcpdump -i eth0 port 22 => to capture packet from the specific port

$ strace -f <command> => to trace systemcall and signal .



$ echo$$ => is the PID of the current process.

$ echo$? => is the return code of the last executed command.

$ echo$# => is the number of arguments in $*

$ echo$* => is the list of arguments passed to the current process
 

$ watch -n.1 “command”=> watch every 10th of second 
Eg:-
$ watch -n.1 “cat /proc/meminfo |grep -i commit”

$ sysctl => configure kernel parameter at runtime
eg :-
$  sysctl -a | grep -i overcommit  => to get value of overcommit of memory

$ ps axo rss,vsz,comm => rss -> actual physical ram used by process
                     Vsz -> virtual ram used by process
                     Comm -> process name .
